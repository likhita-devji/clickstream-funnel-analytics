# Predictive Clickstream Pipeline & E-Commerce Funnel Analytics Engine

## Project Overview

This project builds an end-to-end data analytics pipeline for clickstream data collected from an e-commerce website.

### Features

- ETL Pipeline
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Funnel Analytics
- Customer Abandonment Prediction
- Logistic Regression
- Power BI Dashboard

## Technology Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- SQL
- Power BI
- Git
